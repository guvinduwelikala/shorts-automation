Shorts automation studio

Build a full YouTube Short from one topic.

Enter a topic, generate a script, download portrait footage, render subtitles and audio, then upload the finished video. This UI gives you a clean browser workflow on top of the existing Python pipeline.

Input Topic + privacy
Processing Script, clips, voice, captions
Output Final_Short.mp4 + upload

Workflow

1. Generate Pexels search query. 2. Download portrait clips. 3. Build the short. 4. Add voiceover and subtitles. 5. Upload to YouTube.
